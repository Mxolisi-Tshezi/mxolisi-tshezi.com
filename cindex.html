Data classification Automation on Amazon S3

This project automates the use of Amazon Macie to classify data in specified S3 buckets and notify users of findings. Using a CloudFormation template, the solution enables Macie, performs a classification job, sends results via SNS, and disables Macie based on scheduled triggers. It includes resources like Lambda functions, EventBridge rules, S3 buckets, and SNS notifications to ensure efficient operation. Tech Stack: AWS CloudFormation, Amazon Macie, Lambda, EventBridge, SNS, S3.
